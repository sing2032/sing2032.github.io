Doctype HTML
<!DOCTYPE html> is the first tag added to our html document and it defines the type of document it is.
Headings
The h1, h2, h3... h6 tags are all required to add headings to a web page. The biggest heading is the <h1> and hence the most important one.
Paragraphs
For adding a block of text or a paragraph we can make use of the <p> tag.
Images
The <img> is the tag used to display images. It does not require a closing tag. It uses two main attributes:
src
It provides the source or the path to the image.
alt
It provides alternate text for screen readers and if the image does not load.
Lists
There are three types of lists that can be used:
< ul >
This tag is used to create an unordered list with bullet points.
< ol >
This tag is used to create an ordered list with 1,2,3...
< dl >
This tag is used to create a description list for terms and definitions.
